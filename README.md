# Payment Tracker Project

A full-stack learning project built with **Next.js** (frontend) and **NestJS** (backend) where users can register, login, verify their account via OTP, create online payments using **Stripe**, and view their payment history on a dashboard.

---

## Features

- **User Authentication**
  - Register
  - Login
  - OTP Verification
- **Dashboard**
  - View all payments
  - Create new payments
- **Stripe Integration**
  - Secure online payments
  - Payment records stored in MySQL
- **Database**
  - MySQL with TypeORM for managing users and payments

---

## Tech Stack

- **Frontend:** Next.js 14+ (App Router) + Tailwind CSS
- **Backend:** NestJS + TypeORM + MySQL
- **Authentication:** JWT + OTP verification
- **Payments:** Stripe API
